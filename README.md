# ☕ Velvet Coffee

Projeto de cafeteria desenvolvido para praticar desenvolvimento web com foco em experiência visual, interatividade e responsividade.

O site apresenta um cardápio moderno de cafés, sistema de favoritos, carrinho de compras e um quiz interativo para descobrir qual café combina mais com a personalidade do usuário.

---

## 🚀 Funcionalidades

- Página inicial com banner interativo
- Cardápio de cafés
- Sistema de filtros por categoria
- Busca de cafés em tempo real
- Sistema de favoritos
- Carrinho de compras dinâmico
- Simulação de checkout/pagamento
- Quiz interativo
- Layout responsivo
- Menu mobile
- Tema claro/escuro

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
---

## ⚙️ Funcionalidades em JavaScript

### 📌 Carrinho de Compras
- Adição e remoção de produtos
- Controle de quantidade
- Cálculo automático de subtotal e total
- Armazenamento com `localStorage`

### ❤️ Favoritos
- Salvar cafés favoritos
- Alteração dinâmica do ícone de coração
- Persistência usando `localStorage`

### 🔍 Filtros e Busca
- Filtragem por categorias
- Pesquisa dinâmica pelo nome do café

### 🧠 Quiz
- Quiz interativo para recomendar cafés
- Navegação dinâmica entre perguntas
- Resultado personalizado

### 🌙 Tema Escuro
- Alternância entre modo claro e escuro
- Salvamento automático da preferência
